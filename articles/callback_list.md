# Callbacks

Below is a list of the predefined callbacks that are available in
`mlr3torch`.

| Key                                                                                                     | Label                                   | Packages        |
|:--------------------------------------------------------------------------------------------------------|:----------------------------------------|:----------------|
| [checkpoint](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_checkpoint.html)                     | Checkpoint                              | torch           |
| [history](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_history.html)                           | History                                 | torch           |
| [lr_cosine_annealing](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_cosine_annealing.html)   | Cosine Annealing LR Scheduler           | torch           |
| [lr_lambda](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_lambda.html)                       | Multiplication by Function LR Scheduler | torch           |
| [lr_multiplicative](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_multiplicative.html)       | Multiplication by Factor LR Scheduler   | torch           |
| [lr_one_cycle](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_one_cycle.html)                 | 1cycle LR Scheduler                     | torch           |
| [lr_reduce_on_plateau](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_reduce_on_plateau.html) | Reduce on Plateau LR Scheduler          | torch           |
| [lr_step](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_lr_step.html)                           | Step Decay LR Scheduler                 | torch           |
| [progress](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_progress.html)                         | Progress                                | progress, torch |
| [tb](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_tb.html)                                     | TensorBoard                             | tfevents, torch |
| [unfreeze](https://mlr3torch.mlr-org.com/reference/mlr_callbacks_unfreeze.html)                         | Unfreeze                                | torch           |
