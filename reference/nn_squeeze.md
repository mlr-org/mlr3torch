# Squeeze

Squeezes a tensor by calling
[`torch::torch_squeeze()`](https://torch.mlverse.org/docs/reference/torch_squeeze.html)
with the given dimension `dim`.

## Usage

``` r
nn_squeeze(dim)
```

## Arguments

- dim:

  ([`integer()`](https://rdrr.io/r/base/integer.html))  
  The dimension to squeeze.
